Este projeto foi desenvolvido por Hillary Moura para a disciplina de desenvolvimento web I, para o curso tecnólogo em análise e desenvolvivmento de sistemas.

O projeto consiste em um site de e-commerce de venda de discos, onde o usuário pode navegar por diferentes gêneros musicais, visualizar detalhes dos produtos e adicionar itens ao carrinho de compras. O objetivo do projeto é proporcionar uma experiência de compra online para os amantes de música, permitindo que eles encontrem e adquiram seus discos favoritos de forma fácil e conveniente.

As principais dificuldades encontradas foram:

  • Trabalhar de maneira que o site fosse intuitivo e fácil de usar, já que o uso de CSS não era permitido.
  • Pensar nas páginas de detalhes de cada produto

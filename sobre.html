Este projeto foi desenvolvido para a disciplina de desenvolvimento web I, para o curso tecnólogo em análise e desenvolvivmento de sistemas.

O projeto consiste em um e-commerce de discos e merch, onde o usuário pode navegar por diferentes opções de produtos, visualizar detalhes e adicionar itens ao carrinho de compras..

As principais dificuldades encontradas foram:

  • Trabalhar de maneira que o site fosse intuitivo e fácil de usar.
  • Pensar nas páginas de detalhes de cada produto
